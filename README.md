# 👨‍🍳 Family Chef System

This repository is the central 'brain' for our family meal planning. 

### **How to use**
- **Weekly Menu:** Generate a rolling 2-week plan using the Gemini CLI 'chef' skill.
- **Shopping List:** Use the GitHub Mobile app to view the latest shopping list and check items off.
- **Inventory:** Keep the [FOOD_LIST.md](FOOD_LIST.md) updated with our current freezer surplus.

### **Organization**
- **FOOD_LIST.md**: The master list of dishes and current surplus.
- **MENUS/**: (Future) Storage for our generated and agreed-upon weekly menus.
